# graph-attention
A simple implementation of GRAPH ATTENTION NETWORKS, ICLR 2018
https://arxiv.org/pdf/1710.10903.pdf

# requirements

- Pytorch: 1.0

- Python: 2.7

# datasets

## training

- landmark_clean
